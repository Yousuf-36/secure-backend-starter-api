# Launchstack — Full Stack Prototype

> ⚠️ This is an archived prototype branch.
> The production-ready API boilerplate lives on main.

## What this branch contains
A full-stack prototype built to validate the 
Launchstack architecture. Includes a complete 
Next.js 14 frontend with premium UI, 3D components, 
and a fully integrated FastAPI backend.

## Stack
- Backend: FastAPI + PostgreSQL + SQLAlchemy + JWT
- Frontend: Next.js 14 + TypeScript + Tailwind + R3F
- Auth: JWT + HttpOnly cookies + RBAC
- AI: Provider-agnostic (Groq, Gemini, Ollama)

## Status
Prototype — not maintained. 
See main branch for the production API boilerplate.
